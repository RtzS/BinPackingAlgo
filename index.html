Welcome Here!.

The best fit decreasing and first fit decreasing strategies are among the simplest heuristic algorithms for solving the bin packing problem. They have been shown to use no more than 11/9 OPT + 1 bins (where OPT is the number of bins given by the optimal solution).

Logic

Here bean packing algorithm is implemented using javascript, which is used for solving the problem related adjustment of increasing decreasing square areas. Project

Technology Used

To develope this project HTML5, CSS3 and javascript is used. HTML% & Css is used to give the stylish view, Javascript is used to develop actual logic of project.
